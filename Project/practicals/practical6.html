Ultra Sonic Sensor Range Finder Thickness Measurement Experiment

Aim:

To measure the thickness of an object using an ultra sonic sensor range finder..

Apparatus:

  • Ultra sonic sensor range finder
  • Object to be measured

Procedure:

  1. Place the ultra sonic sensor range finder on one surface of the object to be measured.
  2. Turn on the ultra sonic sensor range finder and adjust the settings as necessary.
  3. Point the sensor towards the opposite surface of the object and trigger the measurement.
  4. Record the measurement displayed on the ultra sonic sensor range finder screen.
  5. Repeat the measurement on different points of the object to get an average reading.
